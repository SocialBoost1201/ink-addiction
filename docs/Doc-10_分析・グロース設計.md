# INK ADDICTION LP – 分析・グロース設計 v1.0

受領日: 2026-02-27  
ステータス: 確定

---

## 1. 目的

本書は、INK ADDICTION LP を「作って終わり」にしないための分析設計および成長戦略を定義する。

本LPの本質はデザインではなく、**問い合わせ数と単価の最大化**である。  
数値で改善できる構造を最初から組み込む。

---

## 2. 計測基盤設計

### 2.1 使用ツール

| ツール                    | 用途                         |
| ------------------------- | ---------------------------- |
| Google Analytics 4        | ページ・イベント計測         |
| Search Console            | 検索クエリ・順位分析         |
| Microsoft Clarity（推奨） | ヒートマップ・セッション録画 |
| 独自イベントログ（DB）    | 問い合わせ質・成約データ     |

### 2.2 計測レイヤー

| レイヤー              | イベント                                                                                                                                                  |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **A. ページレベル**   | `page_view` / `scroll_depth`                                                                                                                              |
| **B. 行動レベル**     | `filter_genre` / `view_work_detail` / `click_availability` / `click_inquiry` / `upload_reference` / `submit_inquiry` / `click_instagram` / `click_tiktok` |
| **C. コンバージョン** | `submit_inquiry`                                                                                                                                          |

---

## 3. コンバージョン設計

### 3.1 主要コンバージョン

```
submit_inquiry（問い合わせ送信完了）
```

### 3.2 マイクロコンバージョン

| イベント           | 意味         |
| ------------------ | ------------ |
| ギャラリー詳細閲覧 | 作品への関心 |
| 空き状況閲覧       | 予約意欲     |
| SNS遷移            | 追いかけ行動 |
| FAQ閲覧完了        | 不安解消完了 |

これらを追跡し、どこで離脱しているかを可視化する。

---

## 4. ファネル設計

### 想定ファネルと目標CVR

| ステップ    | 遷移先          | 目標CVR |
| ----------- | --------------- | ------- |
| 1. Landing  | Gallery閲覧     | 50%     |
| 2. Gallery  | 作品詳細閲覧    | 40%     |
| 3. 作品詳細 | Inquiryクリック | 15%     |
| 4. Inquiry  | 送信完了        | 40%     |

各段階のCVRを GA4 + 独自ログで算出し、ボトルネックを特定する。

---

## 5. ヒートマップ分析

### 5.1 分析項目

- スクロール到達率
- CTAクリック位置
- フォーム離脱箇所
- 滞在時間

### 5.2 改善方針

| 観測                 | 対応           |
| -------------------- | -------------- |
| CTAが見られていない  | 上部へ移動     |
| 価格セクションで離脱 | 価格説明の改善 |
| フォームで離脱       | 入力項目の削減 |

---

## 6. A/Bテスト設計

### 6.1 テスト対象

- ヒーローコピー
- CTA文言
- 価格セクションの位置
- Inquiryフォームのステップ数

### 6.2 実施ルール

| ルール     | 内容               |
| ---------- | ------------------ |
| 同時テスト | 1要素のみ          |
| 期間       | 2週間単位          |
| 採用基準   | 有意差確認後に採用 |

---

## 7. 問い合わせ質分析

### 7.1 分析項目

- 予算レンジの分布
- 希望ジャンルの分布
- 希望日程のパターン
- 成約率（by ジャンル / by 予算）

### 7.2 改善施策

| 観測                   | 対応                       |
| ---------------------- | -------------------------- |
| 高単価ジャンルが少ない | 高単価ジャンルをLP前面表示 |
| 成約率の高い導線がある | その導線を強化             |
| 低質問い合わせが多い   | 事前質問（フォーム）を強化 |

---

## 8. SEO改善サイクル

### 8.1 月次レビュー項目

| 指標           | 確認内容                   |
| -------------- | -------------------------- |
| 検索順位       | 主要キーワードの順位変動   |
| 表示回数       | インプレッション数         |
| CTR            | Click Through Rate         |
| 流入キーワード | 新規・増加キーワードの確認 |

### 8.2 改善施策

| 観測             | 対応             |
| ---------------- | ---------------- |
| CTRが低い        | Titleを改善      |
| 順位が低い       | コンテンツを強化 |
| 表示回数が少ない | 内部リンク追加   |

---

## 9. SNS流入分析

| 計測項目              | 目的             |
| --------------------- | ---------------- |
| Instagramからの流入数 | SNS依存度の把握  |
| TikTok流入数          | SNS依存度の把握  |
| SNS経由の成約率       | 検索流入との比較 |

> **SNS依存度を数値化し、検索流入比率の増加を追う。**

---

## 10. 価格戦略テスト

| パターン | 内容                            |
| -------- | ------------------------------- |
| A        | 明確レンジ表示（例: ¥30,000〜） |
| B        | ぼかし表示（例: 要相談）        |

CVRを比較し、成約率・問い合わせ質に対する影響を検証する。

---

## 11. 空き状況表示効果測定

| 計測内容                  | 目的               |
| ------------------------- | ------------------ |
| Availability閲覧者のCVR   | 表示効果の定量化   |
| 非閲覧ユーザーとのCVR比較 | 空き表示の価値検証 |

> 空き状況表示がCVR向上に寄与しているか検証し、改善判断の根拠とする。

## 13. 成長ロードマップ

| フェーズ    | 施策                                         |
| ----------- | -------------------------------------------- |
| **Phase 1** | CVR最適化（ファネル分析・A/Bテスト）         |
| **Phase 2** | 客単価向上（高単価ジャンル前面化・価格戦略） |
| **Phase 3** | リピート施策導入（顧客管理・再来店促進）     |
| **Phase 4** | TattooBase統合による拡張（多店舗・CRM連携）  |
