## 命名規則

```
Doc-00_Index.md          # インデックス（このファイル）
Doc-01_要件定義.md
Doc-02_サイトマップ.md
Doc-03_ワイヤーフレーム.md
Doc-04_デザイン仕様.md
Doc-05_コンテンツ設計.md
Doc-06_実装仕様.md
```

順番・ファイル名はプロジェクトの進行に合わせて随時更新する。

---

## 更新ルール

- 新しいドキュメントが届いたら、このREADMEのテーブルに追記する
- ステータスは `受領済` / `レビュー中` / `確定` で管理する
- 確定した仕様に対して無断変更は行わない
