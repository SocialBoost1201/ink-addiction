# INK ADDICTION LP – Advanced SEO・GEO ドミネーション設計 v1.0

受領日: 2026-02-27  
ステータス: 確定

---

## 1. 目的

本書は、INK ADDICTIONを以下3つの検索チャネルでトップ表示させるための戦略設計を定義する。

- Google検索（オーガニック）
- ローカル検索（MEO）
- 生成AI検索（GEO: Generative Engine Optimization）

> **目標: 地域名 + タトゥー で上位3位以内**

---

## 2. Local SEO（MEO）完全設計

### 2.1 Google Business Profile 最適化

**必須設定**

| 項目       | 内容                 |
| ---------- | -------------------- |
| カテゴリ   | タトゥーショップ     |
| 副カテゴリ | アートスタジオ       |
| 営業時間   | LP・GBP・NAP完全一致 |
| 写真更新   | 週1回                |
| 投稿       | 月4回                |

### 2.2 口コミ戦略

| 施策           | 内容                                              |
| -------------- | ------------------------------------------------- |
| レビュー依頼   | 施術後LINEでレビュー依頼                          |
| 目標評価       | ★4.8以上を維持                                    |
| 返信           | 全件返信必須                                      |
| キーワード挿入 | 自然な形で挿入（例:「和彫ありがとうございます」） |

### 2.3 地域キーワード設計

地域名 × ジャンルの掛け算でページを強化する。

**例（地域名が横浜の場合）**

```
横浜 タトゥー
横浜 和彫
横浜 アニメ タトゥー
横浜 タトゥースタジオ
```

## 3. E-E-A-T 強化設計

GoogleはE-E-A-T（Experience / Expertise / Authoritativeness / Trustworthiness）を評価する。

### 3.1 Experience（経験）

- 施術工程の写真
- アフターケア動画
- リアルな顧客の声の掲載

### 3.2 Expertise（専門性）

- アーティスト（兄）の経歴詳細
- 受賞歴
- メディア掲載実績
- 修行歴

### 3.3 Authoritativeness（権威性）

- 外部リンクの獲得
- 業界サイトへの掲載
- インタビュー記事

### 3.4 Trustworthiness（信頼性）

- 衛生管理の写真
- 滅菌工程の説明
- キャンセルポリシーの明記
- アーティストの顔写真掲載

## 4. GEO（Generative Engine Optimization）

生成AI（ChatGPT / Gemini / Perplexity等）の回答に引用される構造を設計する。

### 4.1 生成AIに引用される文章構造

- **簡潔な定義文を必ず冒頭に置く**

```
「横浜で和彫を専門に扱うタトゥースタジオです。」
```

- FAQ形式のコンテンツ
- 明確な価格説明
- 具体的な施術工程説明

### 4.2 AIが拾いやすい要素

| 要素         | 実装方針                           |
| ------------ | ---------------------------------- |
| 箇条書き     | 整理された箇条書き形式             |
| 見出し構造   | 論理的なH1〜H3構成                 |
| 定義型文章   | ジャンル・工程・料金を定義文で記述 |
| ローカル情報 | 住所・営業時間・地域名を明示       |

### 4.3 AI回答に引用される条件

- 具体的（数値・固有名詞を含む）
- 簡潔（200文字以内で要点が取れる）
- 信頼性のある情報（公式情報と一致）
- 価格・営業時間が明確に記載されている
