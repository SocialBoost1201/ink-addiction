# INK ADDICTION LP – SEO・コンテンツ戦略設計 v1.0

受領日: 2026-02-27  
ステータス: 確定

---

## 1. 目的

本書は、INK ADDICTION LP を検索流入で継続的に集客できる構造にするためのSEO設計およびコンテンツ戦略を定義する。

タトゥー業界はInstagram依存が強い。しかし検索経由の「検討度の高いユーザー」は成約率が高い。  
**本LPはローカルSEOとジャンルSEOを軸に設計する。**

---

## 2. SEO戦略全体像

### 2.1 集客の3本柱

| 柱                   | キーワード例                                                         |
| -------------------- | -------------------------------------------------------------------- |
| **A. ローカルSEO**   | 地域名+タトゥー / 地域名+和彫 / 地域名+タトゥースタジオ              |
| **B. ジャンルSEO**   | 和彫タトゥー / アニメタトゥーデザイン / ブラックアンドグレータトゥー |
| **C. 不安解消系SEO** | タトゥー痛み / タトゥー料金相場 / タトゥー施術時間                   |

---

## 3. URL設計

### 3.1 基本ルール

- 日本語URLは使用しない
- slugは短く、意味が明確
- 重複ページ作成禁止

### 3.2 主要URL

| パス             | 用途                    |
| ---------------- | ----------------------- |
| `/`              | トップ                  |
| `/styles`        | ジャンル一覧            |
| `/styles/{slug}` | ジャンル詳細（SEOの柱） |
| `/gallery`       | 全作品                  |
| `/gallery/{id}`  | 作品詳細                |
| `/availability`  | 空き状況                |
| `/pricing`       | 料金                    |
| `/safety`        | 衛生管理                |
| `/artist`        | プロフィール            |
| `/faq`           | よくある質問            |
| `/inquiry`       | 問い合わせ              |

## 4. メタ設計

### 4.1 Titleパターン

| ページ   | Title例                                    |
| -------- | ------------------------------------------ |
| トップ   | `地域名 タトゥースタジオ \| INK ADDICTION` |
| ジャンル | `地域名 和彫 タトゥー \| INK ADDICTION`    |
| 作品詳細 | `和彫 龍 タトゥー作品 \| INK ADDICTION`    |

### 4.2 Description設計ルール

- 120文字以内
- 地域名を必ず含める
- 価格帯または強みを含める
- 安心感ワードを入れる

**例**

```
地域名で和彫・洋彫・アニメタトゥーを手掛けるINK ADDICTION。
衛生管理徹底、完全予約制。まずは無料相談から。
```

---

## 5. コンテンツ設計

### 5.1 ジャンルページ構成

各 `/styles/{slug}` ページには以下を必ず含める。

1. ジャンル説明（300〜500文字）
2. 代表作品
3. よくある質問
4. 料金目安
5. 問い合わせ導線

### 5.2 FAQページ設計

- 構造化データ（`FAQPage`）対応
- Q＆A形式
- 最低10問以上

**推奨質問**

| 質問                   |
| ---------------------- |
| 痛みはどれくらい？     |
| 料金はどう決まる？     |
| 施術時間は？           |
| デザイン持ち込み可能？ |
| 未成年は？             |
| カバーアップできる？   |
| アフターケアは？       |

## 7. 画像SEO設計

| ルール       | 内容                                                  |
| ------------ | ----------------------------------------------------- |
| `alt` 属性   | 全画像に必須                                          |
| ファイル名   | 意味のある英単語（例: `japanese-dragon-tattoo.webp`） |
| フォーマット | WebP使用                                              |
| 圧縮         | 必須                                                  |

---

## 8. 内部リンク戦略

| 起点           | リンク先           |
| -------------- | ------------------ |
| ジャンルページ | 作品ページ         |
| 作品ページ     | 同ジャンルの他作品 |
| 作品ページ     | 問い合わせ         |
| FAQページ      | 問い合わせ         |
| 料金ページ     | 問い合わせ         |

> **孤立ページ禁止。すべてのページは必ず内部リンクで繋がっている状態にする。**

---

## 9. 表示速度最適化

| 指標         | 目標                              |
| ------------ | --------------------------------- |
| LCP          | 2.5秒以内                         |
| 画像         | 遅延読み込み                      |
| JavaScript   | 不要なものを削減                  |
| レンダリング | サーバーサイドレンダリング（SSR） |

## 13. NG事項

> [!CAUTION]
> 以下はSEO・コンテンツ品質を損なうため禁止。

- 薄い作品ページの量産
- 重複タイトル
- キーワード詰め込み
- 地域名の乱用
- 自動生成だけの説明文
