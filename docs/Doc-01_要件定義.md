# INK ADDICTION LP – 要件定義 v1.0

受領日: 2026-02-27  
ステータス: 確定

---

## 1. プロジェクト概要

本プロジェクトは、INK ADDICTIONの公式LPを制作し、単なる情報サイトではなく「半DX型スタジオ運営基盤」を構築することを目的とする。

### 目的

- 高単価タトゥー案件の問い合わせ増加
- 問い合わせの質向上
- 予約往復工数の削減
- SNS依存からの脱却
- 将来的なTattooBase統合基盤の確立

---

## 2. 成功指標

### 主要KPI

| KPI              | 目標値     |
| ---------------- | ---------- |
| 問い合わせCVR    | 3〜8%      |
| 有効問い合わせ率 | 70%以上    |
| 予約確定率       | 40%以上    |
| 返信平均時間     | 12時間以内 |

### 副次KPI

- ギャラリー滞在時間
- ジャンルフィルター利用率
- 空き状況閲覧率
- SNS遷移率

---

## 3. ターゲットユーザー

| 区分      | 概要                                       |
| --------- | ------------------------------------------ |
| Primary   | 20〜40代の初回タトゥー検討層。スマホ中心。 |
| Secondary | 経験者、ジャンル検索型ユーザー。           |

## 4. 機能要件

### 4.1 タトゥーギャラリー

- 管理画面から画像アップロード可能
- AI自動ジャンル分類機能
  - 対応ジャンル: 洋彫 / 和彫 / アニメ / ブラック＆グレー / トラディショナル / ファインライン / リアリズム / レタリング
- 管理画面でジャンル手動修正可能
- タグ機能（部位 / サイズ / カラー等）
- ジャンルフィルター表示
- 無限スクロール表示
- 作品詳細ページ表示

### 4.2 スケジュール空き表示（スクリーンショット起点）

**目的:** 予約往復工数の削減

**フロー**

1. 管理者がスケジュールのスクリーンショットをアップロード
2. OCR解析により空き候補抽出
3. 管理画面で確認・編集
4. 承認後公開

**要件**

- スクショアップロード機能
- 空き候補信頼度スコア表示
- 手動編集機能
- 承認制公開
- 問い合わせ時に希望枠選択可能
- 確定は必ず手動

### 4.3 問い合わせフォーム

**必須項目**

| 項目         | 備考 |
| ------------ | ---- |
| 名前         |      |
| 年齢確認     |      |
| 連絡先       |      |
| 希望ジャンル |      |
| 希望部位     |      |
| サイズ       |      |
| カラー       |      |
| 予算レンジ   |      |
| 希望日程     |      |

**任意項目**

- 参考画像アップロード
- 自作デザインアップロード
- カバーアップ写真

**技術要件**

- 複数画像アップロード
- ファイルサイズ制限
- スパム対策
- 自動返信メール
- 管理画面通知

### 4.4 SNS連携

- Instagram導線
- TikTok導線
- ジャンル別投稿リンク設置
- クリックイベント計測

## 6. 非機能要件

### パフォーマンス

- LCP 2.5秒以内
- モバイル優先設計
- 画像最適化

### セキュリティ

- 管理画面認証必須
- アップロードファイル検証
- ログ保存
- XSS / CSRF対策

### スケーラビリティ

- TattooBase統合可能な設計
- 将来的CRM拡張可能

---

## 7. デザイン要件

Apple公式サイトを参照基準とし、以下の方針を守る。

- 余白重視
- タイポグラフィ主導
- 写真主役
- 最小限CTA
- 黒白基調 ＋ アクセントカラー1色
